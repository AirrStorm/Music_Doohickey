# Music Playing Doohickey

A compact, versatile MP3 player that streams audio via Bluetooth (A2DP), plays through wired headphones, or outputs to a speaker. Perfect for portable music projects with a small display.

## Features
- Play MP3s from an SD card
- Stream to Bluetooth A2DP devices
- Wired headphones output
- Built-in speaker output
- Small OLED display for track info
